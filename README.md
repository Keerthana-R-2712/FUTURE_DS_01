# Business Sales Performance Analytics Dashboard

## Overview

This project focuses on analyzing business sales performance using Power BI and Excel. The dashboard provides insights into sales revenue, profit, product performance, regional sales distribution, and shipping trends to support data-driven business decisions.

## Objective

The main objective of this project is to analyze sales data and identify:

* Revenue trends
* Profit performance
* Top-selling products and sub-categories
* Regional sales distribution
* Category-wise sales performance
* Shipping mode effectiveness

## Tools Used

* Power BI
* Microsoft Excel

## Dataset Information

The dataset contains transactional sales records including:

* Order Details
* Product Categories
* Sub-Categories
* Sales
* Profit
* Quantity Sold
* Region
* Ship Mode

## Key Performance Indicators (KPIs)

* Total Revenue: $2.30M
* Total Profit: $286.40K
* Total Quantity Sold: 38K

## Dashboard Features

### Sales Performance Overview

* Total Revenue

### Product Analysis

### Regional Analysis

* Region-Wise Sales Comparison

### Shipping Analysis

* Sales by Ship Mode

### Profitability Analysis

* Sales vs Profit Comparison

### Interactive Filters

* Category Filter
* Region Filter

## Key Insights

### Revenue Performance

* Generated total revenue of $2.30M from business operations.

### Product Performance

* Technology category recorded the highest sales.
* Phones emerged as the top-selling sub-category.

### Regional Performance

* West region achieved the highest sales.
* South region recorded comparatively lower sales.

### Shipping Trends

* Standard Class shipping contributed the highest sales volume.

### Profit Analysis

* Overall profit reached $286.40K, indicating positive business growth.

## Recommendations

1. Increase focus on high-performing Technology products.
2. Expand marketing efforts in lower-performing regions.
3. Optimize inventory for top-selling sub-categories.
4. Promote profitable products to improve margins.
5. Improve logistics strategies based on shipping performance trends.
